# museumplanning.com

Static HTML website for Museum Planning LLC.

**Live site:** museumplanning.com  
**Built with:** HTML + CSS — no framework, no build step, no dependencies  
**Hosted on:** GitHub Pages  

---

## Structure

```
/                        → Home
/work/                   → Case Studies
/services/               → Services
/mvi/                    → Museum Vitality Index
/tools/                  → Planning Tools
/thinking/               → Writing & Research
/thinking/convergence-era/ → Convergence Era essay
/books/                  → Publications
/about/                  → About Museum Planning LLC
/contact/                → Contact
```

## Editing

Each page is a self-contained `index.html` file inside its folder.  
Edit the file → commit → GitHub Pages deploys automatically.

To go live on museumplanning.com: add a `CNAME` file containing `museumplanning.com` and update DNS.
